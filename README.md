# PharmaLens AI

PharmaLens AI is an intelligent pharmaceutical quality control agent powered by Google's Gemini models.

The system analyzes pharmacopoeia monographs and Certificates of Analysis to extract QC tests, generate SOPs, and assist in QC decision making.

## Features

- Pharmacopoeia monograph analysis
- QC test extraction
- Chromatographic parameter detection
- SOP generation
- COA analysis
- AI pharmaceutical assistant

## Technologies

- Python
- FastAPI
- Google Gemini AI
- Google GenAI SDK
- Google Cloud
